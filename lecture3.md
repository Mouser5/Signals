# Конспект лекции №3
1. Декодирование и смежные классы
При передаче через двоичный симметричный канал принятый вектор y = c \oplus e.
Декодирование по максимуму правдоподобия (MLD): поиск вектора ошибки e с минимальным весом Хэмминга \omega(e).
Смежный класс (косет) по вектору a \in F_2^n:

 * Пространство делится на 2^{n-k} косетов по 2^k элементов.
 * Лидер класса — вектор с минимальным весом в косете (наиболее вероятная ошибка).

Свойства:

Алгоритм:
 * Offline: Считаем S = e H^T для всех возможных ошибок. Сохраняем таблицу: синдром \leftrightarrow лидер (ошибка минимального веса). Требуемая память снижается с 2^n до 2^{n-k} ячеек.
 * Online: Принимаем y \to считаем S = y H^T \to находим лидера e по таблице \to восстанавливаем слово \hat{c} = y \oplus e.
